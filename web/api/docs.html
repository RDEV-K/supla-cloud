You can use the Authorize button below to authenticate yourself in the API and use the Try it out buttons to test the endpoints.

You will need to either generate a personal access token and use it as BearerAuth or create a private OAuth application that will enable you to access your account. This is the same flow that your custom applications intergrating with SUPLA should follow.

Beware, that all modifying requests you do here CHANGE the state of your account.
