For the sake of simplicity, all the Try it out forms make requests to the current SUPLA Cloud instance to the /web-api-prefixed endpoints.

This leverages your current web-based authentication and is done solely for the demo convenience. When using the API in your 3rd pary app, you should use the /api/v2.2.0 endpoint prefix.

Beware, that all modifying requests you do here CHANGE the state of your account.
